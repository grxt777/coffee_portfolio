# Coffee House Website

Современный адаптивный сайт для кофейни, созданный с использованием HTML, CSS и JavaScript.

## Особенности
- Адаптивный дизайн
- Интерактивное меню с категориями
- Плавные анимации
- Оптимизированные изображения
- Мобильная версия

## Технологии
- HTML5
- CSS3
- JavaScript (ES6+)
- Font Awesome для иконок
- Google Fonts
